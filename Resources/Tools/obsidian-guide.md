# Obsidian使用指南

## 主题

- `Cmd + ,`-> `外观` -> `主题` -> `选择主题` -> `GitHub Theme`
- 修改外链样式：
  - 查看：`Cmd + Option + I`打开控制台-> 查看链接 css 类->`a.external-link`
  - 修改： `Cmd + ,`-> `外观` -> `CSS代码片段` -> `添加你要修改的样式`
