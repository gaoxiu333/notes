# React Stack

这是一个现代化React技术栈的最佳实践指南，采用Monorepo架构进行项目管理。

## 快速开始

1. 查看 [项目大纲](01-项目大纲.md) 了解整体架构
2. 按照 [初始化指南](01-项目规划/02-初始化指南.md) 进行项目初始化
3. 遵循 [开发规范](03-开发规范/) 进行开发

## 目录结构

```
├── 01-项目规划/        # 项目规划相关文档
├── 02-工程规范/        # 工程化相关规范
└── 03-开发规范/        # 开发相关规范
```

## 技术栈

- 构建工具：Turborepo
- 包管理：pnpm
- 框架：Next.js
- 语言：TypeScript
- 样式：Tailwind CSS
