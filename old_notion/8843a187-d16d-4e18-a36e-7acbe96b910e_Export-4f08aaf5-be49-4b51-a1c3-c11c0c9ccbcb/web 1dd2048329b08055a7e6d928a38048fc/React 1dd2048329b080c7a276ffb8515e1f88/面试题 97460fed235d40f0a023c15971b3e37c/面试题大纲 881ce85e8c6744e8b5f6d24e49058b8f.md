## **BFC**

块级格式化上下文，独立的渲染区域，处于 BFC 内部的元素与外部元素相互隔离，内外元素定位不回相互影响

触发条件

- 根元素
- `position:absolute/fixed`
- `display:inline-block/table`
- `float`
- `ovevflow !== visible` 规则：
- 同一个 BFC 内的元素 `margin`会重叠
- 不会和 `float` 元素区域重叠
- BFC 高度会计算浮动子元素的高度
- 文字层不会被浮动元素覆盖，环绕周围 应用
- 阻止`margin`重叠
- 清除浮动
- 自适应两栏布局
- 阻止元素被浮动元素覆盖
