## 一些基础的依赖

```bash

poetry add fastapi

# uvicorn: 用于运行异步框架
poetry add uvicorn
# starlette: 跨域解决方案，包含在 uvicorn 内

# httpx: 处理高并发场景的 HTTP 客户端
poetry add httpx

# pydantic pydantic-settings: 配置数据类型模型
poetry add pydantic pydantic-settings

# gunicorn: 部署时使用？
```
