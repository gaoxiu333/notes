# 10-projects 项目文件夹

这里存放所有**有明确交付物或截止日期的事务**，包括工作项目、学习项目、实验项目等。

## 子目录结构

按项目性质分类：

- 10-01-前端项目
- 10-02-AI实验
- 10-03-学习项目
- ...

## 项目笔记规范

- 每个项目建立单独文件夹，以`10-XX-项目名称`命名
- 每个项目文件夹下必须有一个同名主笔记，作为项目的入口点
- 项目笔记使用`_tpl-project`模板，必须包含以下元数据：
  - owner: 负责人
  - due: 截止日期
  - status: 项目状态（planning/active/completed/archived）

## 项目生命周期

1. 规划阶段：列出目标和任务分解
2. 执行阶段：跟踪进度和记录问题
3. 完成阶段：总结经验和结果
4. 归档阶段：将已完成项目迁移至`90-archive`
