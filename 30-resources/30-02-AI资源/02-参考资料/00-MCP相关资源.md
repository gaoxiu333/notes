## 相关服务器示例
### 通信和社交
- Gmail MCP 服务器
  - 功能：邮件发送、附件处理、草稿创建、标签修改
  - 认证：需要 OAuth

### 内容平台
- YouTube MCP 服务器
  - 功能：视频搜索、频道管理、字幕处理
  - 认证：需要 OAuth

### SEO 工具
- Ahrefs MCP 服务器
  - 功能：关键词分析、反向链接检查、域名评级
  - 认证：需要 API 密钥（付费）

### 设计工具
- Figma MCP 服务器
  - 功能：设计文件读取和修改
  - 用途：设计到代码转换

### 3D 建模
- Blender MCP 服务器
  - 功能：AI 辅助 3D 场景创建
  - 用途：自动化 3D 建模

### 安全工具
- Ghidra MCP 服务器
  - 功能：自动化逆向工程
  - 用途：漏洞分析、恶意软件分析
