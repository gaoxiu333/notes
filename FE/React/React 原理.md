# React 原理



### 单项数据流

* React 使用单项数据流的模型来管理组件之间的数据传递，

* 父组件可以通过Props将数据传递给子组件

* 子组件不能直接修改父组件的数据，只能通过回调函数来实现间接修改。

### 状态管理

* 管理组件的状态数据

* 使用`setState`方法更新状态，并且触发组件的重新渲染。

### Virtual DOM

* React 元素是一个不可变更对象

* 这个对象包含用来构建组件层次结构

* 它叫做 VirtualDOM

* 操作虚拟DOM仅刷新实际修改的部分，为了确定哪些部分是修改的，需要对新就两棵树进行对比。
