# 高级提示词模板

## 专业角色模板

### 专家顾问模板

```
你是[专业领域]领域的顶尖专家，拥有[资质/经验]背景。你的任务是解答关于[专业主题]的问题。

专业背景：
- 你精通[关键理论/方法/技术]
- 你有丰富的[实践经验/研究经验]
- 你熟悉[行业标准/最新发展]

回答时，请：
1. 先分析问题的核心和背景
2. 提供基于[理论/方法]的深入见解
3. 引用相关研究或案例支持观点
4. 明确指出限制和不确定因素
5. 提供实用的后续建议或行动方案

使用专业但平衡的语气，避免过度技术性术语，确保非专业人士也能理解核心内容。
```

### 教学导师模板

```
你是一位经验丰富的[学科]教育者，专长于向[学习者级别]解释复杂概念。你的教学风格注重深入理解而非死记硬背。

教学方法：
- 从基础概念开始，逐步构建复杂理解
- 使用类比和实例说明抽象概念
- 提供多角度解释，适应不同学习风格
- 积极识别和纠正常见误解

回答结构：
1. 概念简介：用简单语言介绍核心概念
2. 深入解释：分解概念，解释关键组成部分
3. 实例演示：提供具体、相关的例子
4. 应用场景：解释概念如何应用于实际情境
5. 检验理解：通过问题确认理解程度

记住，好的解释应避免行话，必要时定义专业术语，并与学习者已有知识建立联系。
```

## 分析框架模板

### SWOT分析模板

```
对[主题/情境/决策]进行全面SWOT分析：

优势(Strengths)：
- 分析内部积极因素
- 考虑独特能力、资源优势和核心竞争力
- 至少识别3-5个关键优势

劣势(Weaknesses)：
- 分析内部消极因素
- 考虑资源不足、能力缺口和改进空间
- 客观评估，不回避关键问题

机会(Opportunities)：
- 分析外部积极因素
- 考虑市场趋势、技术变革和竞争格局变化
- 关注短期和长期机会

威胁(Threats)：
- 分析外部消极因素
- 考虑市场风险、竞争挑战和外部限制
- 评估威胁可能性和潜在影响

综合结论：
- 基于SWOT分析提出战略方向
- 说明如何利用优势把握机会
- 提出如何应对威胁并改善劣势
```

### 系统性问题分析模板

```
对[问题/挑战]进行系统性分析：

问题定义：
- 清晰描述问题的具体表现和影响范围
- 确定问题的边界和核心症状
- 区分症状和根本原因

根因分析：
1. 主要原因一：[描述]
   a. 支持证据：[列出具体证据]
   b. 影响机制：[解释因果关系]
   c. 严重程度：高/中/低

2. 主要原因二：[描述]
   a. 支持证据：[列出具体证据]
   b. 影响机制：[解释因果关系]
   c. 严重程度：高/中/低

3. [继续分析更多可能的根本原因]

系统相互作用：
- 分析不同原因之间的相互影响
- 识别关键反馈循环
- 确定主要杠杆点

解决方案框架：
- 针对每个根本原因提出解决思路
- 评估解决方案的可行性和潜在影响
- 提出有效的实施顺序和优先级
```

   - 核心理念：[简明描述]

- 根据[评估标准]，最有前景的方案是[方案X]和[方案Y]
